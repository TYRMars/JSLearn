### DOM的本质

* XML

```markup
<?xml version="1.0" encoding="UTF-8"?>
<note>
 <to>Tove</to>
 <from>Jani</from>
 <heading>Reminder</heading>
 <body>Dont forget me this weekend</body>
 <other>
 <a></a>
 <b></b>
 </other>
</note>
```

* HTML

```markup
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Document</title>
</head>
<body>
  <div>
    <p>this is p</p>
  </div>
</body>
</html>
```
